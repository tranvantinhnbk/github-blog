This is a demo of Handwritten Digit Recognition that was built based on TensorFlow JavaScript. The demo runs in your browser, not on a server.

OFFLINE PROCESS:

  • The CNN architecture and source code for training the model are provided here. COLAB

ONLINE PROCESS:

  • The image is loaded from the canvas and preprocessed following the process outlined in the MNIST dataset
  • After that, the image is loaded to the model and a prediction is made
